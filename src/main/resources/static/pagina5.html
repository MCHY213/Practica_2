iOS 17

Esta nueva actualización arranca con una mayor personalización a la hora de editar nuestros propios contactos, ya que se integra una nueva interfaz con fotos y memojis así como una tipografía mucho más llamativa. Estas imágenes aparecerán al momento de recibir una llamada de este contacto y estará de manera constante visible cuando se esté hablando con este contacto. Aquí puedes ver cómo establecer tu póster de contacto en iOS 17.

Si recibes constantes mensajes de verificación de cuentas, ahora iOS 17 tiene la opción de eliminarlos de manera automática. Para activarla, acude a Ajustes, Contraseñas y Opciones de las contraseñas. No tienes más que seleccionar la opción Borrar automáticamente que se encuentra bajo el apartado Códigos de verificación. Por cierto, si quieres saber otra curiosidad, ahora se pueden establecer varios temporizadores al mismo tiempo.
